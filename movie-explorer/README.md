#  Movie Explorer

A simple React application that allows users to search for movies, filter them, and mark favourites. This project demonstrates fundamental React concepts such as state management, conditional rendering, and dynamic UI updates.

##  Live Demo

Deployed on Vercel:
https://react-essentials-assignment-dun.vercel.app/


## Features

* Search movies by **title, genre, or year**
* Display **matching movie results dynamically**
* Add or remove movies from **favourites**
* Display a **favourites list**
* **Dark / Light theme toggle**
* **Responsive design** for desktop and mobile

##  Technologies Used

* React
* JavaScript (ES6)
* CSS3
* HTML5

##  Data Source

The movie data used in this project is **manually created within the project** instead of being fetched from an external API.



##  How to Run the Project

1. Clone the repository

```
git clone https://github.com/GovindgeriNandini/react-essentials-assignment.git
```

2. Navigate to the project folder

```
cd movie-explorer
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm start
```

The app will run at:

```
http://localhost:3000
```
