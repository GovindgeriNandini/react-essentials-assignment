#  Portfolio Card App

A React-based interactive portfolio card that displays a developer profile with skills, description, and interactive features such as theme switching, image navigation, and likes.

##  Live Demo
https://react-essentials-assignm-git-137cad-govindgerinandinis-projects.vercel.app/



##  Features

* Developer **profile card**
* **Profile image carousel**
* **Dark / Light theme toggle**
* **Like button counter**
* **Skills section with interactive tags**
* **Contact information toggle**


##  Technologies Used

* React
* JavaScript (ES6)
* CSS3
* HTML5

##  How to Run the Project

1. Clone the repository

```
git clone https://github.com/GovindgeriNandini/react-essentials-assignment.git
```

2. Navigate to the project folder

```
cd portfolio-card
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm start
```

The app will run at:

```
http://localhost:3000
```
