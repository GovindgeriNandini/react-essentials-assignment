# React Essentials Assignments

This repository contains multiple React projects built while learning **React fundamentals**.

---

## Projects Included

### 1. Movie Explorer

A movie searching application where users can search movies, add them to favourites, and switch between light and dark themes.

**Features**

* Search movies
* Add / remove favourites
* Dark & Light theme toggle
* Responsive design
* Interactive UI

**Live Demo**
https://react-essentials-assignment-dun.vercel.app/

Project Folder: `movie-explorer`

---

### 2. Portfolio Card

A personal portfolio card application that displays profile information, skills, and interactive controls.

**Features**

* Profile image gallery
* Like counter
* Theme toggle (Light / Dark)
* Contact reveal option

**Live Demo**
https://react-essentials-assignment-xxsg.vercel.app/

Project Folder: `portfolio-card`

---
